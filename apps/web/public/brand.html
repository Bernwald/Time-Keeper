Design-Regeln

1. Mobile-first ab 360px. Breakpoints: md: 768px, lg: 1024px.
2. Touch-Targets: min. 44 × 44px auf allen interaktiven Elementen.
3. Safe Area: Bottom-Elemente mit pb-[env(safe-area-inset-bottom)].
4. Farben nur aus Token-System. Keine Ad-hoc #hex Werte.
5. Card-Radius: 16px (rounded-xl). Modals: 24px.
6. Schatten nur aus Token-System: xs / sm / card / md / modal.
7. UI-Texte Deutsch. Code + Kommentare Englisch.
8. Schriften: Fraunces (Display/Headings), DM Sans (Body/UI).
