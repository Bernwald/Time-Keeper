Feature 1

Hybrid Search
mit RRF

Kombiniert Volltext- und Vektorsearch via Reciprocal Rank Fusion – das Beste aus beiden Welten.

-- Volltext (German FTS)
WITH fts AS ( ... websearch_to_tsquery('german') ... ),
-- Vektor-Ähnlichkeit (cosine)
     vec AS ( ... embedding <=> p_embedding ... ),
-- RRF-Score: 1/(60+rank_pos)
   combined AS ( fts FULL OUTER JOIN vec ... )

Feature 3

Entity-Boosted
Search

Fragen werden automatisch gegen Kunden, Projekte und Kontakte aufgelöst – relevante Quellen erhalten einen 1,5×-Boost.

resolveEntities("Müller GmbH")

  boost_factor: 1.5)
